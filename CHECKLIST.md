# ✅ CHECKLIST DE DESPLIEGUE

## 📋 Antes de Empezar
- [ ] Tienes una cuenta en GitHub (si no, crear en [github.com](https://github.com))
- [ ] Tienes una cuenta en Vercel (si no, puedes crearla con tu cuenta de GitHub)
- [ ] Tienes Git instalado en tu computadora
- [ ] Node.js instalado (necesario si quieres testear localmente)

## 🚀 Paso 1: GitHub (15 minutos)
- [ ] Crear repositorio en GitHub llamado `boda-andrea-alberto`
- [ ] Hacer que sea "Public"
- [ ] Clonar repositorio localmente
- [ ] Copiar todos los archivos del proyecto a la carpeta local
- [ ] Ejecutar: `git add .`
- [ ] Ejecutar: `git commit -m "Initial commit"`
- [ ] Ejecutar: `git push origin main`
- [ ] Verificar en GitHub que los archivos estén ahí

## 🎯 Paso 2: Vercel (10 minutos)
- [ ] Ir a [vercel.com](https://vercel.com)
- [ ] Loguearse o crear cuenta con GitHub
- [ ] Click en "Add New..." → "Project"
- [ ] Importar repositorio `boda-andrea-alberto`
- [ ] Cambiar Framework a "Other"
- [ ] Click en "Deploy"
- [ ] Esperar a que termine el despliegue (2-3 min)
- [ ] Ver el enlace: `https://boda-andrea-alberto.vercel.app`

## ✨ Paso 3: Verificar que Funciona
- [ ] Abierto el sitio en línea
- [ ] El sitio se ve bonito (colores, tipografía OK)
- [ ] La navegación funciona (scroll entre secciones)
- [ ] El contador regresivo está contando
- [ ] Las imágenes cargan correctamente
- [ ] El diseño es responsive en móvil

## 🔐 Paso 4: Probar Admin
- [ ] Presionar Ctrl+E en el sitio
- [ ] Entrar con contraseña: `boda2027`
- [ ] Panel admin abre correctamente
- [ ] Puedo editar contenido
- [ ] Puedo guardar cambios
- [ ] Recargar página y los cambios persisten

## 📝 Paso 5: Probar Funcionalidades
- [ ] RSVP: Completar y enviar confirmación
- [ ] Verificar que recibe número de referencia
- [ ] Playlist: Sugerir una canción
- [ ] Verificar que se guarda

## 🌐 Paso 6 (Opcional): Dominio Personalizado
- [ ] Comprar dominio (ej: `boda-aya.com`)
- [ ] Ir a Vercel → Settings → Domains
- [ ] Agregar dominio
- [ ] Configurar DNS en registrador
- [ ] Esperar 24-48 horas para propagación
- [ ] Verificar que `boda-aya.com` resuelve al sitio

## 📊 Paso 7 (Futuro): Integración Emails
- [ ] (Opcional) Integrar SendGrid para emails automáticos
- [ ] (Opcional) Agregar analytics con Google Analytics
- [ ] (Opcional) Agregar formulario de contact
